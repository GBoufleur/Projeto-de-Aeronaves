### EQUIPE CRUSAIR - REPOSITÓRIO OFICIAL

Este repositório será utilizado para centralizar, principalmente os códigos python da disciplina.

## Requisitos do projeto

- Aeronave de transporte de passageiros
- 320 assentos em 2 classes
  
## Missão de projeto:
- Alcance de 8000 nmi, com 320 PAX @ MTOW
- 100 kg por PAX (já contabilizando bagagem)
- Comprimento de pista: 2900m @ MTOW, SL, ISA
- Mach de cruzeiro: 0.85; Máximo Mach Operacional: 0.90
- Reservas: espera (loiter) de 45min + cruzeiro para
alternativa de 200 milhas náuticas

- Base de certificação: FAR 25
